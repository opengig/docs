## Overview

This suite of 4 specialized tools supports the complete workflow for Genome-Wide Association Studies (GWAS), from input preparation to result visualization.

## Input Preparation

- **plan_plink_gwas_inputs** — Plan and validate GWAS input files
- **generate_gwas_sample_data** — Create sample GWAS datasets

## GWAS Analysis

- **run_plink_gwas** — Execute PLINK genome-wide study

## Visualization

- **create_manhattan_plot** — Generate Manhattan plots
