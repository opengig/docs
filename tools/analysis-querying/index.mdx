## Overview

This suite of 8 tools provides comprehensive capabilities for statistical analysis, SQL-based querying, and data exploration of genomic datasets.

## Statistical Analysis

- **analyze_csv** — Perform statistical analysis on CSV data

## SQL Query Tools

- **run_sql_query** — Execute SQL queries on genomic datasets
- **build_sql_query_for_csv** — Generate SQL queries from CSV structure
- **run_sql_query_from_file** — Execute SQL queries stored in files

## Documentation & Profiling

- **describe_csv_columns_with_html** — Generate HTML descriptions of columns
- **describe_csv_columns_with_html_smart** — Generate intelligent HTML descriptions

## Sample Data & Networks

- **make_example_table** — Create example tables for demonstration
- **get_gene_network** — Retrieve gene interaction networks
