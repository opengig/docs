## Overview

This collection of 12 tools handles data conversion between different formats, file processing, and variant filtering operations essential for genomic analysis workflows.

## File Preview & Inspection

- **first_k_lines_of_csv** — Preview the first K lines of a CSV file

## Format Conversion Tools

- **csv_to_vcf** — Convert CSV data to VCF format
- **convert_xlsx_to_csv** — Transform Excel spreadsheets to CSV
- **convert_tsv_to_csv** — Convert tab-separated values to comma-separated values
- **genomic_to_vcf** — Convert genomic data formats to VCF

## Data Creation & Generation

- **create_csv_from_data** — Generate CSV files from structured data

## VCF Operations

- **check_vcf_compatibility** — Validate VCF file compatibility
- **index_vcf** — Create index files for VCF datasets
- **convert_ids_in_csv_using_vcf** — Map and convert identifiers using VCF

## Variant Filtering

- **filter_vcf_by_rsids** — Filter variants by specific RSIDs
- **filter_vcf_by_genes** — Extract variants from specific genes
